# 🧠 Python-MLP

A Python project for implementing a Multi-Layer Perceptron (MLP) neural network, built with Jupyter Notebook. Demonstrates feedforward and backpropagation for classification tasks.

## 📱 About The Project

Python-MLP is a simple and practical implementation of a Multi-Layer Perceptron, a type of artificial neural network with one or more hidden layers. The project uses multiple datasets to train and test the MLP for classification problems.

This project was created for study purposes and to demonstrate the MLP algorithm using Python.

## ✨ Features

- Multi-Layer Perceptron (MLP) implementation
- Feedforward and backpropagation
- Multiple training and testing datasets
